# Quora-Style Post Management System

A Quora-style Post Management System built using Node.js, Express.js, and EJS.  
The application supports full CRUD operations for user-generated posts with RESTful routing and server-side rendering.

---

## Features

- Create, read, update, and delete posts
- REST-based routing using Express.js
- UUID-based unique identifiers for posts
- Method-Override for PUT and DELETE requests
- Server-side rendering using EJS templates
- Middleware-based form handling
- Input validation and basic sanitization

---

## Tech Stack

- Node.js
- Express.js
- EJS
- UUID
- Method-Override
- HTML
- CSS
